1. the style section looks like this:
blockquote{
font: italic 1.2em/1.75em times, serif;
}

2. The size of the em for the font size is 1.2em, and the size of the em for line height is 1.75em. The "em" value is based on the width of the uppercase M in that font.

3. If I specify a font that is not stored on a user's computer, the browser will instead use the default font of the user's computer. Which font that default font is depends on the user's computer.
